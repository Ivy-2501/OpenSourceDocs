教程仅限于已购买订阅用户使用。
软件版权及后续权益属于软件作者。
请确保您已安装最新版的Clash For Android。

Github下载

2.快速导入

此处我们将指导您使用 「 一键导入 」 方式完成配置。
点击下方 导入Clash For Android即可

导入到Clash For Android

3.启动Clash For Android连接

1. 点击上方的一键导入。

2. 点击快速导入会跳转到Clash For Android，把自动更新设置为1440分钟，并保存配置文件。

3. 打开软件主界面，把配置勾选后返回主界面。

4. 开启连接，点击启动，如果是第一次使用会提示你开启VPN连接，点击确定即可。

5. 检查手机状态栏右上角，是否有一个 VPN（钥匙）图标！
